### Clock In/Out Data Structure

There could be 6 kind of Clock In/Out record:

1. Normal Clock In
2. Normal Clock Out
3. Overtime Clock In
4. Overtime Clock Out
5. Rest Clock In
6. Rest Clock Out

You could store 6 columns for data structure but design simple interface with less button for that.
